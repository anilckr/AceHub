# AceHub (Next.js + Tailwind)

## Kurulum
```bash
npm install
npm run dev
```

Tarayıcı:
http://localhost:3000

## Sayfalar
- / (Ana sayfa)
- /urunler
- /hakkimizda
- /referanslar
- /sss
- /sure-sorgulama

## Notlar
- Satın alma ve süre sorgulama şu an demo (front-end). İstersen gerçek ödeme + veritabanı/API entegrasyonu eklenebilir.
